 Expense Tracker Web Application
A simple web-based Expense Tracker built using Flask that allows users to manage their income and expenses, track transactions, and view their financial summary on a dashboard.
 FEATURES
 User registration and login system
 Dashboard NAVIGATION:
   Total income
   Total expenses
Current balance
 Add new transactions (Income or Expense)
 Categorized transactions:
  - Salary
  - Freelance
  - Investment
  - Food & Dining
  - Transportation
  - Utilities
  - Entertainment
  - Shopping
  - Health
  - Education
  - Other
  Transaction descriptions for better tracking
   Delete transactions
   Responsive design for mobile and desktop
    TECH STACK
Backend:Python (Flask)
Frontend:HTML, CSS, JavaScript
Database:  In-memory storage 
Tools: Git, GitHub, VS Code
HOW TO RUN MY PROJECT
1. Ensure Python is installed.
2. Run "python run.py" in your terminal.
3. Open your browser to `http://127.0.0.1:5000'
